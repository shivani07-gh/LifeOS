# Dataset Information

## FaceForensics++ (C23)

This project uses the FaceForensics++ Dataset (C23 version) for training and evaluation.

Dataset Source:
https://www.kaggle.com/datasets/xdxd003/ff-c23

Download Instructions:

1. Download the dataset from Kaggle.
2. Extract the downloaded files.
3. Place the folders inside:

dataset/
├── original/
├── Deepfakes/
├── Face2Face/
├── FaceSwap/
└── NeuralTextures/

Dataset is not included in this repository because of its large size.
